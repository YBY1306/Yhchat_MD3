# 已经实现的功能：
- [x] 手机号/邮箱登录
- [x] 显示会话/管理置顶会话
- [x] 完整解析消息
- [x] Websocket消息实时同步/实时更新会话/草稿同步（目前只能单向同步）
- [x] 发送普通/HTML/Markdown/图片/文件/视频/语音/表情/指令消息（不支持表单指令）
- [x] 解析普通消息/HTML/Markdown/图片/表情/文件消息
- [x] 搜索用户/群聊/机器人，并且添加
- [x] 查看用户/机器人/群聊详情（并且实现protobuf解析）
- [x] 播放语音/下载文件消息，可保存语音到本地
- [x] 查看个人资料
- [x] 修改邀请码/名称/头像
- [x] 修改密码/绑定邮箱
- [x] 搜索会话
- [x] 社区
  - 分区列表
  - 查看关注的分区
  - 查看我发布的文章/已收藏的文章/官方推荐文章
  - 分区文章列表
  - 解析Markdown语法文章
  - 发送普通(或Markdown)类型/评论/分享/点赞/收藏/打赏文章
  - 查看作者/评论者详情
  - 搜索文章/分区
  - 查看分区下绑定的群聊，并且支持申请加入
  - 删除/编辑自己的文章
  - 查看文章详情
  - 屏蔽/取消屏蔽发送文章的用户
  - 分区管理/查看管理分区的关注列表
  - 新建分区/管理分区/删除分区
